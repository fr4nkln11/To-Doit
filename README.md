# To-Doit
Badges go here

To-Doit is a simple To do list app written in Python using the Flask micro framework.

## Technology
* Flask
* Bootstrap
* Bootstrap-icons
* Pretty-checkbox
* Google Fonts
* Languages:
  * Python
  * JavaScript
  * CSS
  * HTML

## Current Features
* Add task to to-do list
* Delete task from to-do list 

## Planned Features
* Multiple color themes
* Update task
* Off-canvas menu
* Animations
