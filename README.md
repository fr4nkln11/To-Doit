To-Doit is a simple To do list app written in Python using the Flask micro framework.

## Technology
* Flask
* Flask-SQLAlchemy
* PostgreSQL
* Vercel
* Railway (Database Provisioning)
* Bootstrap
* Bootstrap-icons
* Pretty-checkbox
* Google Fonts
* Languages:
  * Python
  * JavaScript
  * CSS
  * HTML
  * SQL

## Current Features
* Add task to to-do list
* Delete task from to-do list 

## Planned Features
* New UI components
* re-designed UI
* Multiple color themes
* Update task
* Off-canvas menu
* Animations
